# Calculadora-Basica
Modulo 7
## Descripción
Programa de calculadora básica desarrollado en Python, compuesto por dos archivos:
**Operaciones.py** y **Main.py**.

---

## 📁 Archivos

### Operaciones.py
Módulo que contiene las funciones matemáticas básicas:
- **sumar(a, b)** – Retorna la suma de dos números.
- **restar(a, b)** – Retorna la diferencia de dos números.
- **multiplicar(a, b)** – Retorna el producto de dos números.
- **dividir(a, b)** – Retorna la división de dos números. Maneja el caso de división entre cero.

### Main.py
Archivo principal que interactúa con el usuario:
- Importa el módulo Operaciones.
- Solicita dos números al usuario.
- Muestra un menú de opciones (a/b/c/d).
- Ejecuta la operación seleccionada y muestra el resultado.
- Permite realizar múltiples operaciones y limpia la consola entre cada una.

---

## ▶ ¿Cómo ejecutar?
1. Asegúrate de tener Python instalado.
2. Coloca ambos archivos en la misma carpeta.
3. Ejecuta en la terminal:
   python Main.py
